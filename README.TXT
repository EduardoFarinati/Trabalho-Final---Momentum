README file. Este e um projeto de um jogo baseado na conservacao de momento.
O usuario e uma espaconave e vaga por um labirinto evitando tocar em qualquer
parede, nao ha freios. Deve-se controlar apenas por acelerar em diferentes
orientacoes.
