-----------------------------------------------------------------------------
README file. Este foi um projeto de um jogo baseado na conservação de momento
apresentado para a disciplica de programação orientada a objetos da UFRGS, no
segundo semestre de curso.
O jogador faz o papel de uma espaconave que vaga por um labirinto evitando 
tocar em qualquer parede, mas não há freios. Deve-se controlar apenas por 
acelerar em diferentes orientações.
-----------------------------------------------------------------------------

PROJECT TITLE: Momentum
PURPOSE OF PROJECT: Trabalho Final - Programacao Orientada a Objeto
DEPENDANCIES: Java 7
VERSION or DATE: 1.0
AUTHORS: André Perinazzo, Eduardo Farinati, Marcelo Monaretto
USER INSTRUCTIONS: Utilize ou as setas ou WAD para controlar sua nave:
                   W - acelerar; A - girar para esquerda; D - Girar para direita;
                   Tecla de espaco - especial;
